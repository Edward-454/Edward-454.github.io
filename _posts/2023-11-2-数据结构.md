元组
--
```python

primeNumber = (2,3,5,7)

# 元素全是浮点数的元组
floatNumber = (1.5.1.6.1.7)

```
# 元组索引
```python

```

# 元组查找
in是判断是不是在
```python
number = (1,2,3,4,5)
position = ("东","西","南","中","北")
print(2 in number)
print("中" in position)
```
元组是不可变的
