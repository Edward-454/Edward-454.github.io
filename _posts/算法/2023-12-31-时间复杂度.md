**时间复杂度**
> 用来评估算法运行效率的一个式子


```python
 print('hello world')
```

```python
for i in range(n):
	print('hello world')
```

```python
for i in range(n):
	for j in range(n):
		print('hello world')
```

```python
for i in range(n):
	for j in range(n):
		for k in range(n):
			print('hello world')
```
