**时间复杂度**
> 用来评估算法运行效率的一个式子
```mermaid
graph LR
A[长方形] -- 链接文本 --> B((圆形))
A --> C(圆角矩形)
B --> D{菱形}
C --> D
```

```python
 print('hello world')
```

```python
for i in range(n):
	print('hello world')
```

```python
for i in range(n):
	for j in range(n):
		print('hello world')
```

```python
for i in range(n):
	for j in range(n):
		for k in range(n):
			print('hello world')
```
O(1)  的意思就是   O 就是大约    1 就是单位
