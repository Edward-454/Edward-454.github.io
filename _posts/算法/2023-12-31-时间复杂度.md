**时间复杂度**
> 用来评估算法运行效率的一个式子

O(1)  的意思就是   O 就是大约    1 就是单位  (时间/秒/分/时/天/)
# 案例一
```python
 print('hello world')         
```                  
O(1)
```python
for i in range(n):
	print('hello world')      
```
O(n)
```python
for i in range(n):
	for j in range(n):		 
		print('hello world')
```
o($n^2$)
```python
for i in range(n):
	for j in range(n):
		for k in range(n):     
			print('hello world')
```
o($n^3$)
