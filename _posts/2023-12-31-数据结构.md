变量
--
变量是存储代码的。

变量名 只能由字母、数字、下划线组成。
需要注意的是：
1.不能以数字开头；
2.不能包含空格；
3.大小写敏感、变量A和变量a是不同的。

赋值
--

```python
name =”Downey“
myname = name
print(myname)
```
格式化输出
--

```python
name = "Downey"
print(f"my name is {name}")
print(f"{name}is a good boy")
print(f"{name} likes playing football")

```

--
