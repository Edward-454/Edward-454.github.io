
```python
numberList = [10,20,30,40,50,60,70,80,100]
for number in numberList:

```

for 循环
--
```python
numberList = [10,20,30,40,50,60]
for number in numberList:
	print(number)

# 定义列表fruitList
fruitList = ["apple","grape","cherry"]
# for循环列表fruitList，将元素属性给变量fruit
for fruit in fruitList:
	# 输出变量fruit
	print(fruit)


# 定义列表numberlist
numberlist = [10,20,30,40,50,60]
# for循环列表中的第二项到第六个元素
for number in numberList[1:6]:

# 输出列表的元素
	print(number)
```
